About This Project

Most stock tools (Yahoo Finance, iOS Stocksâ€¦) are built for investors who want to buy or sell shares. But many professionals use stock trends to understand the dynamics of a sector. Existing tools make this difficult because they show only one company at a time and don't allow easy comparison across different timeframes.

This website solves that by offering a simple way to:

  • Select a portfolio of companies
  • View their daily, weekly, and monthly stock performance
  • Compare everything at a glance on one clean dashboard

The tool pulls real-time data from a stock API and presents it through a lightweight website built with HTML, CSS, and JavaScript â€” designed to make trend spotting faster and clearer.
